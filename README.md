# Wild and Free Tools

**350+ free tools. No signup. No paywall.**

A growing library of fast, free web utilities for everyday tasks. Each tool is also exposed as an installable skill for AI coding agents, so you can use them inside your terminal or editor without leaving your workflow.

### [Browse all tools at wildandfreetools.com →](https://wildandfreetools.com)

## For humans

Open any tool in your browser. Convert files, clean text, generate content, edit images, audit SEO, and more. No account needed, no upload limits on most tools, mobile-friendly.

[See the full catalog](https://wildandfreetools.com)

## For AI agents

The same 350+ tools are available as a single installable skill bundle for AI coding agents.

**Works in any of these agents:**
- Claude Code, Claude Desktop
- Cursor, Windsurf
- OpenAI Codex
- Gemini CLI, Gemini Code Assist
- Cline, Roo Code, Continue, Aider
- Goose, Replit Agent
- Any agent that supports Model Context Protocol

**Powered by any of these models:**
- GPT-5, Claude Opus and Sonnet, Gemini
- DeepSeek, Qwen, Llama, Mistral, Grok
- Local models via Ollama, LM Studio, llama.cpp
- Models served by Groq, Together, Fireworks, OpenRouter

If your agent supports Model Context Protocol, every tool in this library works with it. The model behind the agent does not matter.

**Install** (one line):
```bash
npx wild-and-free-tools
```

[Full install guides for each agent →](https://wildandfreetools.com/agents) *(coming soon)*
